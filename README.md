# 🖼️ NFT Preview Card Component

Projeto desenvolvido como parte de um desafio do Frontend Mentor.

## 📌 Sobre o projeto

Este projeto consiste em um card de preview de NFT, com informações como título, descrição, valor em Ethereum e tempo restante, além de um perfil do criador.

O principal foco foi trabalhar layout com Flexbox, organização de código e implementação de interações com hover, incluindo um overlay na imagem.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Variáveis CSS
- Pseudo-classes (:hover)
- Position (relative/absolute)
- Google Fonts

---

## 🎯 O que eu pratiquei

- Estruturação semântica com <main>, <div>, <h1> e <p>
- Organização de arquivos CSS (reset, variables, style, responsive)
- Uso de variáveis CSS para padronização de cores
- Centralização de layout com Flexbox
- Alinhamento de elementos com display: flex
- Criação de overlay com position: absolute
- Efeito hover com transições suaves
- Uso de hsla() para controle de transparência
- Importação e uso da fonte Outfit
